During the sixth week, the internship content focused on deploying applications using a **Container** model on the AWS platform. The primary tasks included understanding the process of containerizing applications using **Docker**, building a Docker Image to deploy to **Amazon Lightsail Container Services**, and configuring the **AWS CLI** using IAM User Access Keys to manage and deploy services from the command line.

### Week 6 Objectives:
* Learn the process of containerizing applications with Docker.
* Build a Docker Image to deploy on Amazon Lightsail Container Services.
* Set up and use the AWS CLI.
* Configure Access Keys for IAM Users to manage AWS resources.
